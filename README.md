# Whatcolor

Whatcolor is a tiny command line tool to give human names
to #RRGGBB hexadecimal sequences.

## Usage

```terminal
whatcolor.py #003353
```

see `whatcolor.py --help` for further information.

## Installation

```bash
git clone https://github.com/mikeevmm/whatcolor
cd whatcolor
bash install.sh
whatcolor --help
```
